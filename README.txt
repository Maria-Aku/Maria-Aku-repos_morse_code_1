Программа для перевода из азбуки Морзе:



Для запуска программы:

1. Необходимо открыть папку debug, расположенную в данной директории.

2. Далее в папке открыть cmd.

3. В командной строке прописать: chcp 1251 (для отображения русских символов).

4. Далее ввести строку: decryption_morse.exe morse.txt decryption.txt.




Получение результата от программы:

Для изменения строки с азбукой Морзе необходимо отредактировать файл morse.txt. Если строка будет с ошибками, они отобразятся на консоли, иначе программа запишет результат в файл decryption.txt, расположенный в той же папке.
